关于本站

手语影像馆 是一个专注于聋人文化与手语影像的站点。

这里收录的内容全部围绕一个主题：用手语讲述故事，用影像记录文化。

📖 站点说明

这是一个静态博客，所有内容存放在 GitHub 上，永久保存——只要 GitHub 还在，这些文章就不会消失。

适合放那些”值得被长期保存”的内容：纪录片介绍、文化科普、深度文章。

🎯 我们的内容方向

  • 🎬 纪录片与短片 — 聋人题材的影像作品介绍与解读
  • 🧏 手语文化 — 手语的历史、演变与社群故事
  • 📝 深度文章 — 关于聋人文化、身份认同与社会议题的思考

每一篇文章都配有视频，让你在看文字的同时，也能用手语”观看”这个世界。

💬 联系与关注

如果你对聋人文化、手语内容感兴趣，欢迎扫码关注我们。一起看见一个更丰富的世界。

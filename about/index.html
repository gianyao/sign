关于本站

手语影像馆，是一个以手语视频为核心的聋人文化影像站点。

我希望这里不只是一个视频合集，而是一座小小的手语影像档案馆：用手语讲述故事，用影像保存记忆，让更多人看见聋人文化里真实、丰富、有力量的一面。

站点说明

本站是一个静态网站，所有内容存放在 GitHub 上，尽可能做到长期保存。

这里每一篇内容的重点不是长篇阅读，而是通过手语视频去观看、理解和感受。

文字部分不是主角，而是视频的补充：它会简单交代背景，整理关键信息，留下必要的说明与延伸，让每一段手语影像都有更清晰的入口。

本站不追热点，不堆文章，只慢慢收藏那些值得留下来的手语视频、聋人故事和文化片段。

内容方向

本站主要收录以下几类内容：

  • 手语视频与影像作品
    收录聋人题材、手语表达、纪录片、短片、访谈等视频内容。

  • 聋人文化与社群故事
    通过视频呈现聋人群体的生活经验、语言表达、身份认同与文化记忆。

  • 文字说明与延伸资料
    用简短文字补充视频背景、人物信息、作品介绍和相关思考，帮助读者更好地进入视频内容。

联系与关注

如果你对手语视频、聋人文化、纪录片或影像记录感兴趣，欢迎关注我们。

这里不只是为了“写文章”，而是想把一些值得留下来的手语影像慢慢收藏起来。

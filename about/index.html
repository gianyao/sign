什么是“影像档案”，什么又是“手语影像馆”？

最近有朋友问我，“手语影像馆”和“影像档案”是什么关系，两个网站有什么区别。

简单来说：

  • sign.yaojian.cc → 叫“影像档案”，是一个 Hexo 博客
  • video.yaojian.cc → 叫“手语影像馆”，是一个视频站点

但真正重要的区别，不在名字上，而在于它们不同的存在方式。

sign.yaojian.cc：永久保存

sign.yaojian.cc 是一个 Hexo 静态博客，所有的文章、图片、配置都存放在 GitHub 上。

这意味着什么呢？

意味着只要 GitHub 还在一天，这个站点就不会真正消失。即使我哪天不续费域名了，所有内容仍然在 GitHub 的仓库里，换个域名就能重新上线。

它的寿命 = 域名续费的时间。

域名一年几十块钱，我可以一直续下去。换句话说，这是一个可以陪伴很久的站。它适合放那些“值得被长期保存”的内容——比如纪录片介绍、文化科普、深度文章。

video.yaojian.cc：有期限的存在

而 video.yaojian.cc 完全不一样。

它跑在我自己购买的一台云服务器上。服务器一年一续费，大约几百到上千块。这不是一笔大钱，但它是一个需要持续决策的事情——每年到了续费的时候，我都会想：还续不续？

如果哪天我不续费了，这台服务器上的所有东西——代码、配置文件、服务——都会在到期后彻底消失。不是“下架”，是真的消失。域名解析不到，数据拿不出来。

它的寿命 = 服务器续费的时间。

所以这个站点更适合放“当下的内容”——正在更新的视频、正在运营的服务。它灵活，可以随时修改和部署，但它也脆弱。

两个站，两种哲学

sign.yaojian.cc video.yaojian.cc
类型 Hexo 静态博客 自建视频站
托管 GitHub Pages 自购云服务器
内容 文章、纪录片介绍 视频播放
续费成本 域名费（~¥60/年） 域名费 + 服务器费（~¥500+/年）
消失风险 极低（GitHub + 域名） 高（不续费即消失）
适合放什么 永久保存的内容 当下的服务

如果把内容比作书：

  • sign.yaojian.cc 是一本真正的书——印出来放在图书馆里，只要图书馆不倒，它就在。
  • video.yaojian.cc 更像一个展览——它活生生的，随时可以更新，但展览结束，一切归零。

所以如果你看到影像档案里的一篇文章，可以放心——它会在那里很久。而如果你在“手语影像馆”上看了一部视频，不妨珍惜当下，因为没有人知道这个站会运行到哪一天。
